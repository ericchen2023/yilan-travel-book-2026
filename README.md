## GitHub Pages 發布

此儲存庫已包含 `.github/workflows/pages.yml`。建立公開儲存庫並將所有檔案放在 `main` 分支後：

1. 開啟 **Settings → Pages**。
2. 在 **Build and deployment** 的 Source 選擇 **GitHub Actions**。
3. 進入 **Actions** 查看 `Deploy static site to GitHub Pages`。
4. 部署完成後，網站網址通常為：`https://<帳號>.github.io/<儲存庫名稱>/`。

網站為純靜態 PWA，所有資源採相對路徑，可部署在專案子路徑。
